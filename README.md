Typogenetic Design
==================

RhinoPython Script for Rhinoceros 6. Provides an interface for interactive architectural optimisation 
using genetic programming for architectural shape generation.

Dependencies
------------

Python 3.6.0
OpenCV 3.4.1 + contrib
Scikit Learn 0.0
Numpy = 1.14.1
Matplotlib 2.2.0
SciPy 1.0.0
