Typogenetic Design
==================

RhinoPython Script for Rhinoceros 6. Provides an interface for interactive architectural optimisation 
using genetic programming for 3D shape generation.

Dependencies
------------

1. Python 3.7.1 32bit
2. OpenCV 4.2.0.34
3. Scikit Learn 0.22.2
4. Numpy = 1.18.3
5. Matplotlib 3.2.1
6. SciPy 1.4.1
