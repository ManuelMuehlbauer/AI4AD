Typogenetic Design
==================

RhinoPython Script for Rhinoceros 7. Provides an interface for interactive architectural optimisation 
using genetic programming for 3D shape generation.

Dependencies
------------

1. Python 3.8.1 64bit
2. OpenCV 4.4.0
3. Scikit Learn 0.23.2
4. Numpy 1.19.1
5. Matplotlib 3.3.1
6. SciPy 1.5.2
